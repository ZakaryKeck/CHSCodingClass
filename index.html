Wordle

Wordle is a game where you have 6 chances to guess a 5 letter word. Each letter gives you some information in the form of a color. Green means that you guessed the letter in the correct spot, yellow means you guessed the letter in the incorrect spot, and gray means the letter is not in the word.

Today we're going to be working together to remake wordle with basic web development.

Open up CodePen here: https://codepen.io/pen/

As you can see in the CodePen there are three basic pieces to every website:

  • HTML - The Structure
  • CSS - The Style
  • Javascript - The Code

Let's start by copy pasting this into the HTML section:

<div>Hello, World!</div>

div is an html element that stands for division. All HTML elements have an opening tag: <div>, and a closing tag </div> that surround its innerHTML.

Now let's add some style to our div. Copy past this into the CSS section.

.awesome-div{ color: red;}

And replace our div in the html section with this:

<div class="awesome-div">Hello, World!</div>

Color is an attribute that will change the color of the text! Now let's add some code. Copy paste this into the JS section: 

        function showMessage() {
          alert("You clicked on a div!");
        }

And replace our div with this:

<div class="awesome-div" onclick="showMessage()">Hello, World!</div>

Sweet! When you click on the div, our message shows up. There are a few pieces to talk about here. onclick is an attribute that you can put on html elements. The value of the attribute is a function that is called when you click on the element. alert() is a function that is built in to web browers and shows the dialogue box. You've probably seen it before!

And here is the final version, with tile animations: 
    checkWord() {
      let guess = this.getEnteredWord();
      let answer = this.word;
      let tileStates = [];
      let letterStates = {};
    
      [...guess].forEach((currentLetter, index) => {
        if (currentLetter === answer[index]) {
          tileStates[index] = "correct";
          letterStates[currentLetter] = "correct";
          guess = guess.substring(0, index) + "-" + guess.substring(index + 1);
          answer = answer.substring(0, index) + "-" + answer.substring(index + 1);
        }
      });
    
      [...guess].forEach((currentLetter, index) => {
        if (currentLetter === "-") {
          //Do nothing
        } else if (answer.includes(currentLetter)) {
          answer = answer.replace(currentLetter, "-");
          letterStates[currentLetter] = "present";
          tileStates[index] = "present";
        } else {
          if (tileStates[index] !== "correct") {
            tileStates[index] = "absent";
          }
          if (letterStates[currentLetter] !== "correct") {
            letterStates[currentLetter] = "absent";
          }
        }
      });
    
      this.activeRow.children[0].classList.add("spin");
      this.animateLetters(this.activeRow.children[0], tileStates, letterStates);
    }
    
    animateLetters(child, tileStates, letterStates) {
      setTimeout(() => {
        this.removeClass(child, "empty");
        this.addClass(child, tileStates[0]);
    
        if (child.nextElementSibling) {
          this.addClass(child.nextElementSibling, "spin");
          this.animateLetters(
            child.nextElementSibling,
            tileStates.slice(1),
            letterStates
          );
        } else {
          for (const [letter, className] of Object.entries(letterStates)) {
            let element = document.getElementById(letter);
            if (className === "correct") {
              this.removeClass(element, "present");
            }
    
            this.addClass(element, className);
          }
          if (this.getEnteredWord() === this.word) {
            this.setScoreInfo(true);
            let scoreInfo = this.getScoreInfo();
            setTimeout(function () {
              alert(
                `Congratulations! Wins: ${scoreInfo.wins}. Current Streak: ${scoreInfo.currentStreak}`
              );
            }, 100);
    
            this.active = false;
          }
    
          if (this.active) {
            this.setNextActiveRow();
          }
        }
      }, 300);
    }

Step 1 - Drawing a Circle

Open this Online IDE we will be using: https://openprocessing.org/. Click that link, and the sketch that opens has some starter code for you. 

function setup() {
  createCanvas(windowWidth, windowHeight);
  background(100);
}

function draw() {
  circle(mouseX, mouseY, 20);
}

setup and draw are two "functions" like we talked about in the powerpoint. They are unique because they are provided for us. setup is run 1 time right after you click play, and draw is run 60 times every second!

Click the play button at the top. A new circle will be drawn over your mouse 60 times each second. mouseX and mouseY are two more variables provided for us. They represent the x-coordinate and y-coordinate of your mouse, respectively. Click the </> icon next to the play button to go back to the code.

createCanvas, background, and circle are three more functions. They create the area where we can draw, color the background, and draw a circle respectively.

Notice how the circle doesn't go away when you move your mouse. What do you think would happen if we move the background(100) out of setup and into draw? 

function setup() {
  createCanvas(windowWidth, windowHeight);
}

function draw() {
  background(100);
  circle(mouseX, mouseY, 20);
}

Next, draw some other circles in various locations, and with various sizes. The parameters for the circle function are (x-coordinate, y-coordinate, circleDiameter).

Step 2 - Circular Motion

When you're making games and animations, you can think of the x axis on a graph as time. As we move forward in time, x increases. frameCount is another variable provided to us that tells us how long the program has been running - it starts at 0 and increases with every frame. Now replace the circle function in your code with this line:

circle(frameCount, 0, 20);

Next, we're going to use a little trigonometry to make the circle move in a circle. Here is an image that illustrates the anatomy of a wave. I like to use https://www.geogebra.org/calculator to visualize functions. Let's visualize sin and cos there.

Next, replace the circle function with this line:

circle(cos(frameCount), 0, 20);

You should be able to see the circle wiggling in the top left corner. How can we increase the size of the circle's movement? Increase the amplitude of the wave times 100!

circle(100*cos(frameCount), 0, 20);

Now the circle is speeding around the top of the canvas. Keep in mind that (0,0) on the canvas is the top left corner - x increases as you move right, and y increases as you move down. So how can we move it into the center? Add vertical and horizontal shift!

circle(100*cos(frameCount) + (width/2), height/2, 20);

Next, we want to slow the circle down. To do that we increase the period of the wave.

circle(100*cos(frameCount/30) + (width/2), height/2, 20);

Lastly, how can we use our knowledge of the two types of waves to create circular motion? Take a look at the sin and cos graphs again. Notice that when cos(x) = 1, sin(x) = 0. We can use this to give us the point (1,0). If you continue through one full period of the waves, you can see the points (1, 0), (0, 1), (-1, 0), (0,-1). The waves compliment each other to create circular motion! Replace your code with this line:

circle(100*cos(frameCount/30) + (width/2), 100*sin (frameCount/30) + (height/2), 20);
